Selenium Java Automation Framework

Automation Testing Framework for an E-Commerce Practice Web Application using Selenium WebDriver, Java, TestNG, and Page Object Model (POM).

---

1.Project Overview

This project automates major functionalities of an e-commerce application including:

- User Login & Logout
- Product Listing
- Add to Cart
- Cart Management
- Checkout Process
- Order Verification
- Form Validations

The framework follows:
- Page Object Model (POM)
- Explicit Wait Strategy
- Data-Driven Testing
- Extent Reports
- Screenshot on Failure

3.Tech Stack

This automation framework is built using modern testing tools and technologies that help in creating reliable, scalable, and maintainable test automation solutions.

Java is used as the core programming language for writing automation scripts.
Selenium WebDriver is used to automate browser actions and perform UI testing.
TestNG helps in organizing and executing test cases efficiently with annotations, reports, and DataProviders.
Maven is used for dependency management and project build automation.
WebDriverManager automatically manages browser drivers without manual setup.
ExtentReports generates detailed HTML execution reports with pass/fail status and screenshots.

These technologies together create a robust automation framework suitable for real-world testing scenarios.
